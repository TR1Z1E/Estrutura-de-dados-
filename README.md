# 💻 Estrutura de Dados - 2026/1

Este repositório contém as implementações e exercícios desenvolvidos durante a disciplina de **Estrutura de Dados**. O foco é o estudo de algoritmos, organização de memória e manipulação de dados utilizando a linguagem Java.

## 📁 Estrutura do Projeto

O repositório está organizado da seguinte forma:

* **`Vetores/`**: Estudos sobre estruturas lineares estáticas e manipulação de arrays unidimensionais.
* **`Matrizes/`**: Implementações envolvendo arrays multidimensionais e lógica de tabelas.
* **`ListaDeExercicios/`**: Resoluções de atividades práticas propostas em aula para fixação de conteúdo.
* **`Prova/`**: Materiais e códigos desenvolvidos para as avaliações da disciplina.

## 🛠️ Tecnologias e Ferramentas

* **Linguagem:** Java 8+
* **Paradigma:** Estrutura de Dados
